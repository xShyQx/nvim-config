My configuration for NeoVim on Linux (not tested on Windows and Mac)

Required packages:
- neovim ( >= 0.9.1 )
- git ( >= 2.19.0 )
- curl or wget
- JDK ( >= 17 ) -- for Java LSP
- python ( >= 3.9 ) -- for Java LSP
- C compiler ( gcc for example )
- libstdc++
- ripgrep ( for using Live Grep in Telescope )
- Nerd Font ( optional )
